# Фитнес с Катей Усмановой — тестовое (3 экрана)

Повтор трёх экранов лендинга [usmanovafit.gymteam.ru/mainpage](https://usmanovafit.gymteam.ru/mainpage),
максимально близко к оригиналу (цвета, типографика, композиция), но на чистой семантической БЭМ-вёрстке.

## Что сделано
1. **Hero** — заголовок капсом, оффер, CTA-кнопка, фото Кати.
2. **Достижения** — секция «Доверьте свое тело чемпионке» с карточкой достижений
   и галереей результатов «до/после» (горизонтальный скролл).
3. **Подбор программы** — секция «Не знаете, с чего начать?» с рабочей формой:
   маска телефона, валидация, состояния ошибки и успеха.

Шапка (sticky + бургер-меню) и футер — как обвязка.

## Соответствие оригиналу
- Палитра из оригинала: розовый `#f66297`, голубой акцент слов `#7ea2f3`, фон `#f3f3f3`.
- Шрифт `Gilroy` (проприетарный) с бесплатным фолбэком `Manrope`.
- Изображения — реальные ассеты с официальной страницы.

## Как работает интерактив
- Бургер-меню открывает/закрывает навигацию на мобильном.
- «Подробнее» на карточке скроллит к форме и подставляет выбранную программу.
- Телефон форматируется маской `+7 (999) 123-45-67`.
- Сабмит валидирует имя и телефон; при ошибке — подсветка и подсказки, при успехе — подтверждение. Данные логируются в консоль (бэкенда в тестовом нет).

## Технологии
- Чистые **HTML + CSS + JS**, без фреймворков и сборки.
- Вёрстка по **БЭМ**, семантические теги (`header/main/section/article/form/footer`).
- Резиновая/адаптивная вёрстка: `clamp()`, CSS Grid `auto-fill/minmax`, контрольные точки 960 / 768 / 420 px.
- Доступность: `aria-*`, видимый фокус, `prefers-reduced-motion`.

## Структура
```
index.html
assets/
  css/style.css
  js/main.js
  img/            обложки программ и портрет
```

## Запуск локально
Открыть `index.html` в браузере или поднять статический сервер:
```bash
npx serve .
```

— Инструмент: Claude Code (вайбкодинг). Время сборки: ~1 час.
